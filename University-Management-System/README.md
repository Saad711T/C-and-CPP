# University management system
Simple university management system built with C++ and OOP techniques.

## Features
- Search students by ID
- Display student information

## Project Structure

- `UniversityManagementSystem.cpp` : Source code for all project
- `Students.txt` : Simple text database for students (Dummy data not for real persons).
